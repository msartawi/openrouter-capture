# Support

Use GitHub Discussions for usage questions and discovery methodology. Use GitHub Issues for reproducible defects and approved feature work.

Do not post:

- Router passwords or session tokens.
- Complete capture trees, HARs, or configuration backups.
- Public IP addresses or private information about household devices.
- Vendor firmware unless redistribution is authorized.
- Exploit instructions or authentication bypasses.

For security issues, follow `SECURITY.md`.

Product questions about the desktop app belong in [OpenRouterDesk](https://github.com/msartawi/openrouterdesk), not here.
